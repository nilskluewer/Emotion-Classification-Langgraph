# Coding Sheet for Qualitative Content Analysis (Mayring)*

*Research Question:* To what extent can an LLM encapsulate the Theory of Constructed Emotions to analyse emotions in a context-related manner?

## 1. Contextual Sensitivity

*Description:* Captures whether and how interviewees perceive that the LLM analysis considers the context (social, political, situational) when recognizing emotions.

*Coding Labels:*
• *Context Considered:* The LLM analysis is perceived as context-sensitive.
• *Context Ignored:* The LLM analysis is perceived as not context-sensitive.
• *Context Partially Considered:* The LLM analysis is perceived as partially context-sensitive.

## 2. Granularity of Emotion Recognition

*Description:* Captures whether interviewees perceive the LLM analysis as differentiated in emotion recognition or whether it is limited to simple categorizations.

*Coding Labels:*
• *Analysis Differentiated:* The LLM analysis is perceived as differentiated and nuanced in emotion recognition.
• *Analysis Undifferentiated:* The LLM analysis is perceived as undifferentiated and limited to simple categories.
• *Analysis Partially Differentiated:* The LLM analysis is perceived as partially differentiated.

## 3. Utility for Moderation
*Description:* Captures how useful interviewees consider the LLM analysis for the moderation of online discussions.

*Coding Labels:*
• *Useful for Moderation:* The LLM analysis is assessed as useful for moderation.
• *Not Useful for Moderation:* The LLM analysis is assessed as not useful for moderation.
• *Partially Useful for Moderation:* The LLM analysis is assessed as partially useful for moderation.
